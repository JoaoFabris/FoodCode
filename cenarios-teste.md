# Cenários de Teste - FoodCode App

## Cenário 1: Navegação na Home
**Objetivo**: Verificar se a navegação principal funciona corretamente
**Passos**:
1. Acessar página inicial
2. Verificar carregamento de produtos
3. Navegar entre categorias
4. Verificar elementos da interface

**Resultado esperado**: Todos os elementos carregam e navegação funciona

## Cenário 2: Busca de Produtos
**Objetivo**: Validar funcionalidade de busca
**Passos**:
1. Inserir termo de busca
2. Verificar resultados filtrados
3. Testar busca vazia
4. Testar caracteres especiais

**Resultado esperado**: Busca retorna resultados corretos

## Cenário 3: Carrinho de Compras
**Objetivo**: Testar fluxo completo do carrinho
**Passos**:
1. Adicionar produtos ao carrinho
2. Modificar quantidades
3. Remover produtos
4. Verificar cálculo total

**Resultado esperado**: Carrinho funciona corretamente

## Cenário 4: Login de Usuário
**Objetivo**: Validar processo de autenticação
**Passos**:
1. Preencher formulário de login
2. Validar campos obrigatórios
3. Submeter dados
4. Verificar redirecionamento

**Resultado esperado**: Login realizado com sucesso

## Cenário 5: Gestão de Produtos (Admin)
**Objetivo**: Verificar funcionalidades administrativas de produtos
**Passos**:
1. Acessar painel admin
2. Listar produtos
3. Filtrar por categoria
4. Editar/excluir produtos

**Resultado esperado**: Gestão de produtos funciona

## Cenário 6: Gestão de Usuários (Admin)
**Objetivo**: Testar administração de usuários
**Passos**:
1. Visualizar lista de usuários
2. Filtrar por tipo (admin/cliente)
3. Buscar usuários
4. Executar ações

**Resultado esperado**: Sistema de usuários operacional

## Cenário 7: Gestão de Pedidos (Admin)
**Objetivo**: Validar controle de pedidos
**Passos**:
1. Listar pedidos
2. Filtrar por status
3. Alterar status dos pedidos
4. Ver detalhes

**Resultado esperado**: Fluxo de pedidos funciona
